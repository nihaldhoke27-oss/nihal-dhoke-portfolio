## About This Portfolio

This is a single-page portfolio website built with vanilla HTML, CSS, and JavaScript. It showcases my work in B2B SaaS content strategy and LinkedIn organic growth research.

### Features

- ⚡ Pure HTML/CSS/JS — zero dependencies, zero build step
- 🎨 Dark mode with glassmorphism and gradient accents
- ✨ Scroll-triggered reveal animations
- 💬 Typing effect on the hero subtitle
- 📊 Animated stat counters
- 🔢 Animated skill bars
- 🖱️ Cursor glow effect on insight cards
- 🌐 Orb parallax on mouse move
- 📱 Fully responsive with mobile hamburger menu
- 🚀 GitHub Pages deployable — no build required

## Deploy to GitHub Pages

1. Push this repo to GitHub as `nihal-dhoke-portfolio`
2. Go to **Settings → Pages**
3. Set Source to **Deploy from a branch → main → / (root)**
4. Site will be live at `https://nihaldhoke27-oss.github.io/nihal-dhoke-portfolio`

---

## Local Development

No build step required. Just open `index.html` in a browser:

```bash
# Option 1: Direct open
open index.html

# Option 2: With live-server (VS Code extension or npm)
npx live-server .
```
